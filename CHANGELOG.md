# 更新日志

所有重要的项目更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且此项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-XX

### 新增
- 初始版本发布
- 基于 AntV G6 的关系图可视化库
- 支持节点展开功能
- 支持自定义节点和边样式
- 支持多种布局算法配置
- 内置工具栏（缩放、自适应、重置）
- 支持拖拽、缩放、选择等交互
- 完整的 TypeScript 类型定义
- 支持事件回调（节点点击、展开、图表就绪）
- 提供 React 组件示例
- 完整的 API 文档和使用示例

### 技术特性
- 使用 ES6+ 模块化开发
- 支持 UMD 格式打包
- 完整的 webpack 构建配置
- 支持 Babel 转译
- 提供开发和生产环境配置

### 文档
- 完整的 README 文档
- API 接口文档
- 使用示例和最佳实践
- TypeScript 类型定义
- 在线演示页面
